## VS code

### Enable Emmet support for react
In the settings add the following:

```json
  "emmet.includeLanguages": {
    "javascript": "javascriptreact"
  }
```

## Create project

```sh
npx create-react-app <projectName>
```

## React router dom
```sh
npm install react-router-dom
```
